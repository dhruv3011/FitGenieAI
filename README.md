# 🧠 AI Fitness Coach – Android App

An AI-powered fitness coach built with **Jetpack Compose** that generates structured workout and diet plans in a clean, chat-based UI.

---

## 🚀 Features

* 💬 Chat-based AI interaction
* 📊 Structured responses:

  * Goal
  * Workout Plan
  * Diet Plan
  * Tips
* 🧩 Card-based UI for better readability
* ✨ Smooth animations (section reveal)
* 🥗 Beginner-friendly & customizable plans

## 🧠 How It Works

1. User enters a fitness-related query
2. AI generates a structured response
3. App parses response into sections
4. UI renders clean cards with animations

## 🧑‍💻 Getting Started

1. Clone the repo:

```bash
git clone https://github.com/dhruv3011/FitGenieAI.git
```

2. Open in **Android Studio**

3. Add your API key in `local.properties`:

```
GEMINI_API_KEY=your_key_here
```

4. Run the app 🚀
